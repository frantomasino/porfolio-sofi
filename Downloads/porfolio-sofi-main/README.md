# Portfolio Arquitecta — Studio Arch

## Cómo abrir el portfolio

**Doble clic en `index.html`** → se abre en el navegador. No necesita nada más.

---

## Panel de Administración

- Hacé clic en **"Studio Arch"** en la barra lateral derecha
- Contraseña: `arch2025`
- Desde ahí podés subir fotos PNG para cada proyecto y editarlos

---

## Cómo cambiar la contraseña

Abrí `index.html` con un editor de texto (VS Code, Notepad++) y buscá esta línea:

```
const ADMIN_PASS = 'arch2025';
```

Cambiá `arch2025` por la contraseña que quieras.

---

## Cómo publicarlo online (gratis)

1. Entrá a [netlify.com](https://netlify.com)
2. Arrastrá la carpeta entera al dashboard
3. Listo — te da una URL pública en segundos

---

## Notas sobre las imágenes

- Las fotos se guardan en el navegador (localStorage)
- Formato recomendado: PNG o JPG, máximo 5MB por imagen
- Si cambiás de navegador o computadora, hay que volver a subir las fotos
- Para que las fotos persistan en cualquier dispositivo, se necesita un hosting online
