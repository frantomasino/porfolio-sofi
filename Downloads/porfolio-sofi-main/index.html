☁️ Configuración de almacenamiento en la nube Las fotos se suben a Cloudinary (gratuito, 25GB). Editá las dos primeras líneas del <script> con tu CLOUDINARY_CLOUD_NAME y CLOUDINARY_UPLOAD_PRESET. Sin eso, las fotos se guardan temporalmente como antes pero no persisten entre dispositivos.

Pasos: 1) Creá cuenta en cloudinary.com → 2) Dashboard → copiá Cloud Name → 3) Settings → Upload → Add upload preset (Unsigned) → copiá el nombre → 4) Pegalo en el código.

Nuestro proceso parte siempre del entendimiento profundo del lugar, el cliente y el programa. Creemos en la conversación prolongada antes del primer trazo.
